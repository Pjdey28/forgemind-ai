# 💻 ForgeMind AI Frontend

The frontend delivers an intuitive and modern user experience for interacting with ForgeMind AI. Built with Next.js and Tailwind CSS, it provides engineers with AI-powered document search, operational dashboards, knowledge graph visualization, and maintenance intelligence.

---

# 🎯 Responsibilities

* Authentication
* Dashboard
* Document Upload
* AI Chat
* Knowledge Graph Visualization
* Compliance Dashboard
* Maintenance Insights
* User Settings
